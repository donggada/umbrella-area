## 🧩 기능 구현

- 기상청 데이터 API 사용하여 일기예보 기능구현
- Kakao Map API 사용하여 현재위치 및 우산대여존을 지도로 구현
- 우산 대여 기능 구현
- 우산 대여 구독권 구매 기능 구현 예정
- 관리자 페이지에서 지도에 우사대여존 지정 할수있도록 구현 예정

## 🛠️ 사용 기술 및 라이브러리

- Spring-boot
- MySQL
- JPA
- 기상청데이터 API
- Kakao Map API
